# Wazuh Agent Integration (DVWA EC2)

## 🎯 Objective

Connect a vulnerable EC2 instance to Wazuh Manager for monitoring

## ⚙️ Setup

* Installed Wazuh agent on DVWA EC2
* Configured manager IP
* Opened ports 1514, 1515

## 🔗 Communication

Agent successfully connected to Wazuh Manager

## 🧠 Outcome

Centralized logging is enabled for attack detection

## 🔐 Security Insight

Proper agent-manager communication is critical for real-time monitoring
